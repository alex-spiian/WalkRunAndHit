# WalkRunAndHit

## Realized:
Transitions between animations "Idle", "Walk", "Run", "Hit";  
Movement behavior.

Hero control:  
"W" - go forward;  
"S" - go back;  
"D" - go to the right;  
"A" - go to the left;  
"W + left shift" - run;  
"Space" - hit with a sword;
