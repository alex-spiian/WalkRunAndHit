# WalkRunAndHit

## Realized:
Transitions between animations "Idle", "Walk", "Run", "Hit";  
Movement behavior.

Hero control:  
"W" - to walk forward;  
"S" - to walk back;  
"D" - to rotate to the right;  
"A" - to rotate to the left;  
"W + left shift" - to run;  
"Space" - to hit with a sword;
